# LoList
The app for saving your todo and other staff

## Config

Frontend config located in `.svelte.env`

Server config located in `.env`

## Before start

Migration:
```node_modules/.bin/sequelize db:migrate```

Seeds:
```node_modules/.bin/sequelize db:seed:all ```

## How to start:

- dev:
`npm run dev`
- prod
`npm start`
